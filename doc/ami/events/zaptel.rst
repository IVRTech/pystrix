Zaptel
======

Zaptel is a legacy PRI-interaction technology, deprecated by DAHDI. Support for this feature-set
will likely disappear in the future.

Members
-------

All of the following objects should be accessed as part of the `ami.zaptel_events` namespace,
regardless of the modules in which they are defined.
